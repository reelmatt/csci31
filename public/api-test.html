Film API test

This HTML/JS page tests the Film API by listing all; then creating, finding, updating, and deleting a film entry using the REST API. A few notes:

  • The form below contains the two pieces of information asked for when logging a film through the app.
  • After entering a film and rating, click the "Test the API" button below to run the tests.
  • Log output will be displayed in the browser console. For the update step, the rating is changed to "5" -- it is recommended you enter in a different value in the form to see the change.
  • See the console for output as the process completes.
